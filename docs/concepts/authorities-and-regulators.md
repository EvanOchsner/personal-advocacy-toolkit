# Authorities and regulators

*(Stub — authored in Phase 3A, expanded in Phase 5.)*

The "who cares about this?" map. For each situation type, lists candidate
authorities (state insurance departments, state AG consumer-protection
units, CFPB, FTC, medical boards, state bars, licensing boards, HUD, EEOC,
local police / DA consumer-fraud units) and what each actually does with a
complaint. Data lives in `data/authorities.yaml`.
