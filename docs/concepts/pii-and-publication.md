# PII and publication

*(Stub — authored in Phase 3C, expanded in Phase 5.)*

When and how to publish derivatives of a case file — what to scrub (names,
emails, phones, VINs, policy / claim / account numbers, addresses, EXIF
GPS, PDF text layers under visual redactions, docx metadata), what to
preserve (the narrative shape and the documentary integrity of exhibits),
and what post-checks verify the scrubbing actually worked. The failure
mode this section exists to prevent: a "redacted" PDF whose text layer
still contains the redacted content.
