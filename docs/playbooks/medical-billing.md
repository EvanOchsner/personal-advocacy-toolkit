# Playbook: medical billing

*(Stub — authored in Phase 5.)*

> **Reference material, not legal advice.**

Covers: surprise-billing protections (No Surprises Act), balance billing,
EOB / CPT code reading, itemized-bill requests, hospital financial-
assistance policies, state AG consumer-protection units, state insurance
departments (for plan-side issues), medical boards (for provider-side
issues), and the CFPB where collections start.
