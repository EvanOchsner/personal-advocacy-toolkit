# Tutorial 05: Going public safely

*(Stub — authored in Phase 5.)*

When and how to publish a derivative of your case: PII scrubbing, PDF
redaction (text-layer removal, not just visual), docx metadata scrub,
EXIF scrub, git history sanitization, and the post-checks that catch
silent misses. Emphasizes the "Option 1" guidance: ship a template +
narrative, not the raw file tree.
